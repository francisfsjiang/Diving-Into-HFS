\chapter*{摘要}
\label{sec:abstract}
\vspace*{-10mm}

本文对Hadoop File System 0.21的核心源代码进行了分析,
并依此解释了HFS的设计思想, 同时给出了软件架构图, 详细类图和核心过程的顺序图,
指出了系统实现与设计思想的呼应之处,
并以源代码中的相关代码为例证明了我们的判断。

更进一步的, 本文描述了团队对Hadoop云计算平台与文件系统之间的关系,
并对课程设计提供的早期分析报告进行修正, 给了出完整的分析过程与修订后的报告。
